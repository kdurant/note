# 稳定排序
待排序的记录序列中可能存在两个或两个以上关键字相等的记录。排序前的序列中`Ri`领先于`Rj`（即i<j）.若在排序后的序列中`Ri`仍然领先于`Rj`，则称所用的方法是稳定的。
